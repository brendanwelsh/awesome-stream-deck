# Contribution Guidelines

Thanks for taking the time to contribute! Please read these guidelines before opening a pull request so we can keep this list awesome.

## Adding to the list

- Make an individual pull request for each suggestion.
- Add your entry at the bottom of the most appropriate category.
- Use the following format: `[Name](link) - Description.`
- Keep descriptions short, clear, and free of marketing language.
- The description must start with an uppercase character and end with a period.
- Check your spelling and grammar.
- Don't mention `Stream Deck` in the description; the whole list is about it.
- New categories, or improvements to the existing ones, are welcome.

## What makes a good entry

This is a curated list of the *best* resources, not an exhaustive index. Only suggest projects that you have used or can personally recommend. An entry should be:

- **Awesome.** Genuinely one of the best options in its category.
- **Maintained.** Avoid abandoned, archived, or deprecated projects. Evergreen resources (such as icon packs and hardware designs) are an exception.
- **Documented.** It should have a working link and a clear description of what it does.
- **Relevant.** It must relate to the Elgato Stream Deck or compatible control surfaces.

## Updating your pull request

If a maintainer asks you to revise your pull request, please do so. It often takes a few iterations to get an entry right, and that's perfectly normal.

## Removing an entry

If you find a link that is broken, a project that has been abandoned or archived, or content that no longer belongs, please open a pull request to remove it and explain why.

Thank you for your suggestions!
